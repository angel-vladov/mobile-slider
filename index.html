    • Mobile Slider

    Converts a list of items into a slider (carousel) when viewed on mobile devices

    Project maintained by angel-vladov Hosted on GitHub Pages

    Key Features

    • Integration with Bower
    • Modifies mobile look, leaves desktop as it is
    • No additional elements added that change your DOM elements tree depth
    • Unobtrusive mobile styles that apply when you need them
    • Configure it your way: inline markup or via JavaScript

    Mobile Friendly, Desktop Neutral

    Open on a mobile device or resize the page to be less than 1024px and the mobile slider will be applied. ( breakpoint is configurable )
